SETUP

================
Directory
================
Copy and paste the entire TARS directory to the designated host
server (e.g. Local host)

================
Database
================
Reset your MySQL Password to a password of your choice before you proceed.
Go into db.php in the main folder and change the constant DATABASE_PATH to where the database will be hosted.
Also change the DATABASE_PASSWORD constant to the password you chose for MySQL.
1. Open up phpmyadmin
2. Click "New" in the left hand bar.
3. Enter "TARS" as the database name.
4. Click "Create"
5. Select the empty "TARS" database.
6. Click the "Import" tab
7. Click "Choose File"
8. Select "TARS.sql"
9. Scroll down and click "Go"
